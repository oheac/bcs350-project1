# RUBRIC (100 POINTS TOTAL)

## Content (40 points)
- Business concept clarity (10 pts)
- Completeness of answers (10 pts)
- Quality and relevance of written content (10 pts)
- Organization and structure (10 pts)

## Design (40 points)
- Visual consistency and aesthetics (10 pts)
- Effective color and font choices (10 pts)
- Layout and usability (10 pts)
- Accessibility and responsiveness considerations (10 pts)

## Interactivity (20 points)
- Appropriate use of interactive elements (10 pts)
- Functionality and usability (10 pts)

# Submission Requirements:

### How to submit:
* Everything is to be posted on GitHub and the website need to be hosted on [GitHub Pages](https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site)
* Submit only the GitHub Repo and Website URL link on Brightspace

What to include in the repo:
- Web Design Document: this will be the README.md file in your repo. This includes :
	- all the questions as well as your answers
	- any external resources used (links)
	- wireframe design (hand-drawn/computer-aided)
	- main sections to include:
		+ project overview
		+ target audience
		+ content strategy
		+ information organization
		+ visual desing (wireframe and more)
		+ interaction / functionality
		+ technical overview
		+ timeline / project milestone
- The website files (HTML, CSS, JS)
	- Code must be original and well-commented
